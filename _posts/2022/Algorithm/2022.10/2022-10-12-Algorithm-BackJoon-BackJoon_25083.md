## 풀이

- " \ " 을 입력할때는 "\"한개를 더 넣어줘야한다.
  = " " " 을 중간에 입력할때는 " "\ "을 입력해야한다.

```java
public class Back_25083 {
    public static void main(String[] args) {

        System.out.println("         ,r'\"7");
        System.out.println("r`-_   ,'  ,/");
        System.out.println(" \\. \". L_r'");
        System.out.println("   `~\\/");
        System.out.println("      |");
        System.out.println("      |");
    }
}



```
